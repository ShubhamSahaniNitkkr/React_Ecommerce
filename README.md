## Purpose

A phone-store e-commerce demo with cart, product details, and PayPal sandbox checkout.

## Use Cases

- Shopping cart state management
- Product catalog UI
- Payment integration prototyping

## Tech Stack

- React 16
- React Router v5
- Context API
- Bootstrap 4
- PayPal SDK

## How to Run Locally

```bash
npm install && npm run dev
```

> On Node 17+, if build fails: `export NODE_OPTIONS=--openssl-legacy-provider`
